# Промпт для новой редакции договора

Нужно подготовить новую редакцию договора для сайта New Generation English на основе файла `assets/documents/new-generation-service-contract.docx` и юридического ревью `CONTRACT_LEGAL_REVIEW.md`.

Задача: сделать договор спокойным, понятным для родителя/ученика и более устойчивым юридически, без тяжелого канцелярита.

Основные требования:

1. Переименовать договор в договор индивидуальных репетиторских/консультационных услуг по английскому языку, если не предполагается выдача документов об образовании, квалификации или прохождении государственной аттестации.
2. В предмете договора убрать обещание формальной образовательной программы и оставить индивидуальную работу по английскому языку: занятия, консультации, диагностика, материалы, обратная связь.
3. Отдельно развести роли `Заказчик` и `Обучающийся`. Для несовершеннолетнего указать, что Заказчик является законным представителем или действует с согласия законного представителя.
4. Добавить блок про дистанционный формат: ProgressMe, кабинет ученика, видеосвязь/ссылки, техническая готовность ученика, сбои связи.
5. Уточнить абонемент: количество занятий, длительность, частота, срок действия пакета, порядок продления, перенос остатка занятий.
6. Уточнить оплату: предоплата, чек самозанятого через `Мой налог`, способ оплаты по согласованным реквизитам, право не проводить занятие при отсутствии оплаты.
7. Переписать отмены и переносы: срок уведомления, письменные каналы связи, неявка, опоздание, технические проблемы, перенос по инициативе Исполнителя.
8. Уточнить возврат: что возвращается только за неоказанные занятия, срок возврата, формула расчета, корректировка чека при возврате.
9. Добавить блок про результат: Исполнитель отвечает за добросовестную работу, маршрут, материалы и обратную связь, но не гарантирует конкретный балл, оценку, уровень CEFR, поступление или иной результат.
10. Усилить авторские права: материалы только для личного обучения, нельзя публиковать, пересылать третьим лицам, продавать или выдавать за свои.
11. Добавить персональные данные: данные используются только для организации обучения, диагностики, отчетов, материалов, расписания и связи по занятиям; для взрослых и детей есть отдельные согласия.
12. Добавить порядок претензий и коммуникации: Telegram/WhatsApp/e-mail для рабочих уведомлений, e-mail для претензий, срок ответа.
13. Вычистить форматирование: склеенные пункты, нумерацию раздела 3, реквизиты Заказчика, единый стиль статуса Исполнителя.

Стиль: простой русский язык, уважительный тон, без угроз и тяжеловесных юридических фраз. Договор должен защищать Исполнителя, но не пугать нормального клиента.
