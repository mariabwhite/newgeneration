What you will do in this lesson

≈ 3 min

In this lesson, you will learn to talk about places, rooms, and objects using "there is" and "there are". You will describe what exists, ask questions, and make negatives.

Start

You will look at a room or workspace and name the things you can see.

Route

First learn there is / there are, then make questions and negatives.

Finish

At the end you will describe a place in 6–8 sentences and ask 3 questions.

03
Language Focus

There is / There are

≈ 8 min
FormSingular (one thing)Plural (two or more)
PositiveThere is a desk.
There is an orange notebook.		There are two books.
There are some pencils.
NegativeThere isn't a TV.
There isn't a printer.		There aren't any toys.
There aren't many chairs.
QuestionIs there a lamp?
Is there a laptop?		Are there any plants?
Are there two windows?
Short answerYes, there is. / No, there isn't.Yes, there are. / No, there aren't.
Rule: Use there is for one thing. Use there are for two or more things. After a question, say Yes, there is / No, there isn't or Yes, there are / No, there aren't.

Typical mistake

Wrong: There are a desk. / Is there two chairs?
Right: There is a desk. / Are there two chairs?

Alex's room

Alex has a small room. There is a bed near the window. There is a desk next to the bed. There are two books on the desk. There are some pencils in a cup. There isn't a TV in the room. There aren't many toys on the floor. Alex likes his room because it is clean and bright.

  6. 06Are there pencils or notebooks?
